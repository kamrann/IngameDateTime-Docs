IngameDateTime > Ingame Clock > Get Clock State

Get Clock State

Returns a representation of the internal clock state, which can be stored/saved and later used to restore a clock to the same time.

WARNING: Specifying bIncludeTimers is experimental and unsupported. It most definitely is not safe to use with a SaveGame.

Inputs

Target
Ingame Clock Reference

Include Timers
Boolean

EXPERIMENTAL

Outputs

Return Value
Ingame Clock State Structure

The current clock state
