IngameDateTime_Dev > Ingame Clock > Adjust Realtime Scale

Adjust Realtime Scale

Changes the clock's realtime scale on the fly.
The scale value is the number of fundamental calendar units in one realtime second.

NOTE: Will have no effect if the clock property bDynamicRealtimeScale is not set to true.

Inputs

In
Exec

Target
Ingame Clock Object Reference

New Scale
Float

New realtime scale to apply to the clock

Outputs

Out
Exec
