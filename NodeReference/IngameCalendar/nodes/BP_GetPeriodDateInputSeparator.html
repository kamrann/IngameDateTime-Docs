Get Period Date Input Separator

Returns the symbol used as an input separator for dates/times, preceding the specified period.

For example, periods Month and Day may be preceded by a '/', whilst Minute and Second could use a ':'.

Inputs

Target
Ingame Calendar Object Reference

Period Id
Name

ID of period

Outputs

Return Value
String

Input separator for dates/times
