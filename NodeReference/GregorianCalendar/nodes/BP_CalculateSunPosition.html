Calulate Sun Position

Calculates the position of the Sun for a given latitude, longitude and time.

Inputs

Target
Gregorian Calendar Object Reference

Latitude
Float

Degrees
Longitude
Float

Degrees
Utc Time
Ingame Date Time Value Structure (by ref)

Regardless of longitude, the input time is assumed to be UTC

Outputs

Elevation
Float

Angle above horizon, in degrees
Azimuth
Float

Angle from north, in degrees
